# Building AI Agents with Helicone

Learn how to monitor multi-step AI agent workflows, track tool calls, and debug complex agent behavior using Helicone's observability features.

## What You'll Learn

In this guide, you'll learn how to:
- Track agent sessions and conversation flows
- Monitor tool calls and function executions
- Debug agent decision-making processes
- Analyze agent performance and costs
- Implement error handling for agents

## Prerequisites

<Note>
  This guide assumes you have:
  - Basic understanding of AI agents and LangChain/LlamaIndex
  - Helicone account and API key
  - An agent application to monitor
</Note>

## Agent Session Tracking

### Setting Up Session Tracking

Track your agent's entire conversation flow using session IDs:

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import OpenAI from "openai";
    
    const client = new OpenAI({
      baseURL: "https://ai-gateway.helicone.ai",
      apiKey: process.env.HELICONE_API_KEY,
    });
    
    // Create a unique session for this agent interaction
    const sessionId = `agent-${Date.now()}`;
    
    async function runAgent(userQuery: string) {
      // Step 1: Analyze the query
      const analysis = await client.chat.completions.create({
        model: "gpt-4o",
        messages: [{ role: "user", content: `Analyze: ${userQuery}` }],
        headers: {
          "Helicone-Session-Id": sessionId,
          "Helicone-Session-Name": "Query Analysis",
          "Helicone-Session-Path": "/agent/analyze",
        },
      });
      
      // Step 2: Execute actions
      const execution = await client.chat.completions.create({
        model: "gpt-4o",
        messages: [{ role: "user", content: `Execute: ${analysis}` }],
        headers: {
          "Helicone-Session-Id": sessionId,
          "Helicone-Session-Name": "Action Execution",
          "Helicone-Session-Path": "/agent/execute",
        },
      });
      
      return execution;
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from openai import OpenAI
    import time
    
    client = OpenAI(
        base_url="https://ai-gateway.helicone.ai",
        api_key=os.getenv("HELICONE_API_KEY"),
    )
    
    # Create a unique session for this agent interaction
    session_id = f"agent-{int(time.time())}"
    
    def run_agent(user_query: str):
        # Step 1: Analyze the query
        analysis = client.chat.completions.create(
            model="gpt-4o",
            messages=[{"role": "user", "content": f"Analyze: {user_query}"}],
            extra_headers={
                "Helicone-Session-Id": session_id,
                "Helicone-Session-Name": "Query Analysis",
                "Helicone-Session-Path": "/agent/analyze",
            },
        )
        
        # Step 2: Execute actions
        execution = client.chat.completions.create(
            model="gpt-4o",
            messages=[{"role": "user", "content": f"Execute: {analysis}"}],
            extra_headers={
                "Helicone-Session-Id": session_id,
                "Helicone-Session-Name": "Action Execution",
                "Helicone-Session-Path": "/agent/execute",
            },
        )
        
        return execution
    ```

## Tracking Tool Calls

### Monitor Function Executions

Track when your agent calls external tools or functions:

```typescript
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "https://ai-gateway.helicone.ai",
  apiKey: process.env.HELICONE_API_KEY,
});

async function agentWithTools(query: string) {
  const tools = [
    {
      type: "function",
      function: {
        name: "search_database",
        description: "Search the product database",
        parameters: {
          type: "object",
          properties: {
            query: { type: "string" },
          },
        },
      },
    },
  ];
  
  const response = await client.chat.completions.create({
    model: "gpt-4o",
    messages: [{ role: "user", content: query }],
    tools,
    headers: {
      "Helicone-Session-Id": "agent-tools",
      "Helicone-Property-Agent-Type": "tool-calling",
    },
  });
  
  // Tool calls are automatically tracked in Helicone
  return response;
}
```

## Debugging Agent Behavior

### View Agent Flow in Dashboard

1. Navigate to **Sessions** in the Helicone dashboard
2. Find your agent session by ID or name
3. View the hierarchical flow of all requests
4. Inspect inputs, outputs, and tool calls
5. Check token usage and costs per step

### Analyze Agent Performance

Query agent metrics using the API:

```bash
curl -X POST https://api.helicone.ai/v1/session/query \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "filter": {
      "properties": {
        "Agent-Type": {
          "equals": "ReAct"
        }
      }
    },
    "limit": 100
  }'
```

## Error Handling for Agents

### Implement Fallbacks

```typescript
async function robustAgentCall(query: string) {
  try {
    return await client.chat.completions.create({
      model: "gpt-4o,claude-sonnet-4", // Fallback to Claude
      messages: [{ role: "user", content: query }],
      headers: {
        "Helicone-Session-Id": sessionId,
        "Helicone-Retry-Enabled": "true",
      },
    });
  } catch (error) {
    // Log error with custom property
    console.error("Agent failed:", error);
    throw error;
  }
}
```

## Common Agent Patterns

### ReAct Agent Pattern

```typescript
async function reactAgent(task: string) {
  const sessionId = `react-${Date.now()}`;
  let step = 0;
  
  while (step < 5) {
    // Reasoning step
    const thought = await client.chat.completions.create({
      model: "gpt-4o",
      messages: [{ role: "user", content: `Think about: ${task}` }],
      headers: {
        "Helicone-Session-Id": sessionId,
        "Helicone-Session-Path": `/react/step${step}/thought`,
      },
    });
    
    // Action step
    const action = await client.chat.completions.create({
      model: "gpt-4o",
      messages: [{ role: "user", content: `Act on: ${thought}` }],
      headers: {
        "Helicone-Session-Id": sessionId,
        "Helicone-Session-Path": `/react/step${step}/action`,
      },
    });
    
    step++;
  }
}
```

### Multi-Agent Collaboration

```typescript
async function multiAgentSystem(task: string) {
  const sessionId = `multi-agent-${Date.now()}`;
  
  // Agent 1: Researcher
  const research = await client.chat.completions.create({
    model: "gpt-4o",
    messages: [{ role: "user", content: `Research: ${task}` }],
    headers: {
      "Helicone-Session-Id": sessionId,
      "Helicone-Property-Agent-Role": "researcher",
      "Helicone-Session-Path": "/multi/research",
    },
  });
  
  // Agent 2: Writer
  const content = await client.chat.completions.create({
    model: "gpt-4o",
    messages: [{ role: "user", content: `Write based on: ${research}` }],
    headers: {
      "Helicone-Session-Id": sessionId,
      "Helicone-Property-Agent-Role": "writer",
      "Helicone-Session-Path": "/multi/write",
    },
  });
  
  // Agent 3: Reviewer
  const review = await client.chat.completions.create({
    model: "claude-sonnet-4",
    messages: [{ role: "user", content: `Review: ${content}` }],
    headers: {
      "Helicone-Session-Id": sessionId,
      "Helicone-Property-Agent-Role": "reviewer",
      "Helicone-Session-Path": "/multi/review",
    },
  });
  
  return review;
}
```
