## Overview

## Cost Calculation

Helicone automatically calculates costs based on:

### Token-Based Costs

- **Input tokens**: Cost of tokens in the prompt
- **Output tokens**: Cost of tokens in the completion
- **Cached input tokens**: Discounted cost for cached prompts (Anthropic, OpenAI, etc.)
- **Thinking tokens**: Cost for reasoning tokens (o1, o3 models)
- **Cache write tokens**: Cost for writing to prompt cache (5min and 1hr TTLs)

### Modality Costs

- **Image tokens**: Vision model input/output costs
- **Audio tokens**: Audio model input/output costs
- **Video tokens**: Video model processing costs
- **File tokens**: Document processing costs

### Additional Costs

- **Web search**: Search API call costs (Perplexity)
- **Request cost**: Per-request pricing (some providers)

### Cost Breakdown Structure

```typescript
interface CostBreakdown {
  inputCost: number;              // Standard input token cost
  outputCost: number;             // Standard output token cost
  cachedInputCost: number;        // Cached token cost (discounted)
  cacheWrite5mCost: number;       // Cost to write 5min cache
  cacheWrite1hCost: number;       // Cost to write 1hr cache
  thinkingCost: number;           // Reasoning token cost (o1, o3)
  webSearchCost: number;          // Search API costs
  requestCost: number;            // Per-request fees
  
  // Per-modality breakdowns
  image?: ModalityCostBreakdown;
  audio?: ModalityCostBreakdown;
  video?: ModalityCostBreakdown;
  file?: ModalityCostBreakdown;
  
  totalCost: number;              // Sum of all costs
}

interface ModalityCostBreakdown {
  inputCost: number;
  cachedInputCost: number;
  outputCost: number;
}
```

```typescript Node.js
const response = await fetch("https://api.helicone.ai/v1/metrics/totalCost", {
```

## Cost Optimization Strategies

<Tab title="Model Selection">
Use cheaper models for simpler tasks:

```typescript
function selectModel(taskComplexity: "simple" | "medium" | "complex") {
  const models = {
    simple: "gpt-4o-mini",      // $0.15/$0.60 per 1M tokens
    medium: "gpt-4o",            // $2.50/$10 per 1M tokens
    complex: "claude-3-opus",    // $15/$75 per 1M tokens
  };
  
  return models[taskComplexity];
}

// Simple task - use cheaper model
const response = await client.chat.completions.create({
  model: selectModel("simple"),

});
```

<Tab title="Prompt Caching">
Use prompt caching to reduce costs for repeated prompts:

```typescript
// Anthropic prompt caching
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
  baseURL: "https://anthropic.helicone.ai",
  defaultHeaders: {
    "Helicone-Auth": `Bearer ${process.env.HELICONE_API_KEY}`,
  },
});

const response = await client.messages.create({
  model: "claude-3-5-sonnet-20241022",
  max_tokens: 1024,
  system: [
    {
      type: "text",
      text: "You are a helpful assistant...", // This will be cached
      cache_control: { type: "ephemeral" },
    },
  ],
  messages: [
    { role: "user", content: "What is 2+2?" },
  ],
});

// Subsequent requests reuse cached prompt at 90% discount
// Input: $3/MTok -> Cached: $0.30/MTok
```
</Tab>

<Tab title="Token Optimization">
Reduce token usage with concise prompts:

```typescript
// Expensive - verbose prompt
```

<Tab title="Smart Caching">
Cache responses for repeated queries:

```typescript
const cache = new Map<string, string>();

async function getCachedCompletion(prompt: string) {
  // Check cache first
  if (cache.has(prompt)) {
    console.log("Cache hit - $0 cost");
    return cache.get(prompt);
  }
  
  // Cache miss - make API call
  const response = await client.chat.completions.create({
    model: "gpt-4o-mini",
    messages: [{ role: "user", content: prompt }],
  });
  
  const completion = response.choices[0].message.content;
  cache.set(prompt, completion);
  
  return completion;
}

// 100% cost savings on cache hits
```

## Cost Alerts

### Set Budget Alerts

Create alerts when costs exceed thresholds:

```typescript
interface BudgetAlert {
  name: string;
  threshold: number;  // in dollars
  period: "day" | "week" | "month";
  filters?: any;
}

async function checkBudgetAlert(alert: BudgetAlert) {
  const startTime = getPeriodStart(alert.period);
  
  const response = await fetch("https://api.helicone.ai/v1/metrics/totalCost", {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${process.env.HELICONE_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      filter: alert.filters || {},
      timeFilter: {
        start: startTime.toISOString(),
        end: new Date().toISOString(),
      },
    }),
  });
  
  const { data: cost } = await response.json();
  
  if (cost >= alert.threshold) {
    await sendAlert({
      title: `Budget Alert: ${alert.name}`,
      message: `Cost $${cost.toFixed(2)} exceeded threshold $${alert.threshold}`,
      severity: "warning",
    });
  }
}

// Check alerts periodically
const alerts: BudgetAlert[] = [
  { name: "Daily Budget", threshold: 100, period: "day" },
  { name: "Monthly Budget", threshold: 3000, period: "month" },
  { name: "Production Costs", threshold: 500, period: "day", filters: {
    request_response_rmt: {
      properties: { "Environment": { equals: "production" } }
    }
  }},
];

setInterval(() => {
  alerts.forEach(alert => checkBudgetAlert(alert));
}, 60 * 60 * 1000); // Check every hour
```

```typescript
async function generateMonthlyInvoice(userId: string, billingPeriod: { start: Date; end: Date }) {
  // Get user's costs
  const costResponse = await fetch("https://api.helicone.ai/v1/metrics/totalCost", {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${process.env.HELICONE_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      filter: {
        request_response_rmt: {
          user_id: { equals: userId },
        },
      },
      timeFilter: {
        start: billingPeriod.start.toISOString(),
        end: billingPeriod.end.toISOString(),
      },
    }),
  });
  
  const { data: cost } = await costResponse.json();
  
  // Get request count
  const countResponse = await fetch("https://api.helicone.ai/v1/request/count/query", {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${process.env.HELICONE_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      filter: {
        request_response_rmt: {
          user_id: { equals: userId },
        },
      },
    }),
  });
  
  const { data: requestCount } = await countResponse.json();
  
  // Calculate markup (e.g., 2x cost)
  const markup = 2.0;
  const customerPrice = cost * markup;
  
  return {
    userId,
    period: billingPeriod,
    requests: requestCount,
    cost: cost,
    price: customerPrice,
    margin: customerPrice - cost,
  };
}
```

<Tab title="Cost Anomaly Detection">
**Scenario**: Detect unusual cost spikes.

```typescript
async function detectCostAnomalies() {
  // Get last 30 days of costs
  const response = await fetch("https://api.helicone.ai/v1/metrics/costOverTime", {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${process.env.HELICONE_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      filter: {},
      timeFilter: {
        start: new Date(Date.now() - 30 * 24 * 60 * 60 * 1000).toISOString(),
        end: new Date().toISOString(),
      },
      dbIncrement: "day",
      timeZoneDifference: 0,
    }),
  });
  
  const { data: dailyCosts } = await response.json();
  
  // Calculate average and standard deviation
  const costs = dailyCosts.map(d => d.cost);
  const avg = costs.reduce((a, b) => a + b, 0) / costs.length;
  const stdDev = Math.sqrt(
    costs.map(c => Math.pow(c - avg, 2)).reduce((a, b) => a + b, 0) / costs.length
  );
  
  // Check if today's cost is anomalous (> 2 std devs)
  const todayCost = costs[costs.length - 1];
  if (todayCost > avg + 2 * stdDev) {
    await sendAlert({
      title: "Cost Anomaly Detected",
      message: `Today's cost $${todayCost.toFixed(2)} is ${((todayCost / avg - 1) * 100).toFixed(0)}% above average`,
      severity: "critical",
    });
  }
}
```
