---
title: 'Query Evaluations'
api: 'POST https://api.helicone.ai/v1/evals/query'
description: 'Query evaluation results for your requests'
---

## Overview

Retrieve evaluation scores and results for requests within a specified time range.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="filter" type="object" required>
  Filter criteria for evaluations (use `{}` for all)
</ParamField>

<ParamField body="timeFilter" type="object" required>
  Time range filter
  
  <Expandable title="time filter properties">
    <ResponseField name="start" type="string" required>
      Start time in ISO 8601 format
    </ResponseField>
    <ResponseField name="end" type="string" required>
      End time in ISO 8601 format
    </ResponseField>
  </Expandable>
</ParamField>

<ParamField body="offset" type="number" default="0">
  Number of evaluations to skip for pagination
</ParamField>

<ParamField body="limit" type="number" default="10">
  Maximum number of evaluations to return
</ParamField>

<ParamField body="timeZoneDifference" type="number">
  Timezone offset in minutes from UTC
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of evaluation objects
  
  <Expandable title="evaluation object">
    <ResponseField name="id" type="string">
      Evaluation ID
    </ResponseField>
    <ResponseField name="request_id" type="string">
      Associated request ID
    </ResponseField>
    <ResponseField name="name" type="string">
      Evaluation metric name
    </ResponseField>
    <ResponseField name="score" type="number">
      Evaluation score
    </ResponseField>
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

```bash cURL
curl --request POST \

  --header 'Authorization: Bearer YOUR_API_KEY' \

  --data '{
    "filter": {},

    },

  }'
```

```python Python
import requests

url = "https://api.helicone.ai/v1/evals/query"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

payload = {
    "filter": {},
    "timeFilter": {
        "start": "2024-01-01T00:00:00Z",
        "end": "2024-03-02T23:59:59Z"
    },
    "offset": 0,
    "limit": 10
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()
```
