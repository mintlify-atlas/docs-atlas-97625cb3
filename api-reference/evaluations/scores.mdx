---

## Overview

Retrieve a list of all unique evaluation metric names used in your organization.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of evaluation score names (strings)
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://api.helicone.ai/v1/evals/scores \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

```python Python
import requests

url = "https://api.helicone.ai/v1/evals/scores"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
data = response.json()
```

```javascript JavaScript
const response = await fetch('https://api.helicone.ai/v1/evals/scores', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
  },
});

const data = await response.json();
```

</RequestExample>
