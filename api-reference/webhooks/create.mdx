## Overview

Create a webhook to receive real-time notifications when requests are logged to Helicone.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="destination" type="string" required>
  Webhook destination URL (must be HTTPS)
</ParamField>

<ParamField body="config" type="object" required>
  Webhook configuration options
  
  <Expandable title="config properties">
    <ResponseField name="sampleRate" type="number" default="100">
      Percentage of requests to send (0-100)
    </ResponseField>
    <ResponseField name="propertyFilters" type="array" default="[]">
      Array of property filters to match
      
      Example:
      ```json
      [
        {"key": "environment", "value": "production"},
        {"key": "user_tier", "value": "premium"}
      ]
      ```
    </ResponseField>
  </Expandable>
</ParamField>

<ParamField body="includeData" type="boolean" default="true">
  Include full request/response data in webhook payload
</ParamField>

## Response

<ResponseField name="data" type="object">
  Created webhook data
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.helicone.ai/v1/webhooks \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "destination": "https://example.com/webhook",
    "config": {
      "sampleRate": 100,
      "propertyFilters": [
        {
          "key": "environment",
          "value": "production"
        }
      ]
    },
    "includeData": true
  }'
```

```python Python
import requests

url = "https://api.helicone.ai/v1/webhooks"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

payload = {
    "destination": "https://example.com/webhook",
    "config": {
        "sampleRate": 100,
        "propertyFilters": [
            {"key": "environment", "value": "production"}
        ]
    },
    "includeData": True
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()
```


## Webhook Payload

When a request matches your webhook filters, Helicone sends a POST request to your destination URL:

```json
{
  "request_id": "req_abc123",
  "created_at": "2024-03-02T10:30:00Z",
  "model": "gpt-4",
  "provider": "OPENAI",
  "user_id": "user_456",
  "cost": 0.0045,
  "latency": 1250,
  "status": 200,
  "properties": {
    "environment": "production"
  },
  "request_body": {...},
  "response_body": {...}
}
```

## Webhook Security

<Note>
  All webhook payloads are signed with HMAC-SHA256. Use the `hmac_key` from the response to verify webhook authenticity.
</Note>
