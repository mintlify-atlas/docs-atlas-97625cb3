---
title: 'Delete Webhook'
api: 'DELETE https://api.helicone.ai/v1/webhooks/{webhookId}'
description: 'Delete a webhook configuration'
---

## Overview

Remove a webhook from your account to stop receiving notifications.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Path Parameters

<ParamField path="webhookId" type="string" required>
  The unique identifier of the webhook to delete
</ParamField>

## Response

<ResponseField name="data" type="null">
  Returns null on success
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url https://api.helicone.ai/v1/webhooks/wh_abc123 \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

```python Python
import requests

url = "https://api.helicone.ai/v1/webhooks/wh_abc123"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.delete(url, headers=headers)
```

```javascript JavaScript
const response = await fetch(
  'https://api.helicone.ai/v1/webhooks/wh_abc123',
  {
    method: 'DELETE',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
    },
  }
);
```

</RequestExample>
