---
title: 'Session Metrics'
api: 'POST https://api.helicone.ai/v1/session/metrics/query'
description: 'Get aggregated metrics for sessions'
---

## Overview

Retrieve aggregated metrics across sessions including cost, token usage, and performance statistics.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="nameContains" type="string" required>
  Filter sessions by name substring (use empty string for all)
</ParamField>

<ParamField body="timezoneDifference" type="number" required>
  Timezone offset in minutes from UTC
</ParamField>

<ParamField body="pSize" type="string">
  Percentile size for statistics: `p50`, `p75`, `p95`, `p99`, or `p99.9`
</ParamField>

<ParamField body="useInterquartile" type="boolean">
  Use interquartile range for outlier filtering
</ParamField>

<ParamField body="timeFilter" type="object">
  Time range filter
  
  <Expandable title="time filter properties">
    <ResponseField name="startTimeUnixMs" type="number">
      Start time in Unix milliseconds
    </ResponseField>
    <ResponseField name="endTimeUnixMs" type="number">
      End time in Unix milliseconds
    </ResponseField>
  </Expandable>
</ParamField>

<ParamField body="filter" type="object">
  Advanced filter criteria
</ParamField>

## Response

<ResponseField name="data" type="object">
  Session metrics object
  
  <Expandable title="metrics properties">
    <ResponseField name="total_sessions" type="number">
      Total number of sessions
    </ResponseField>
    <ResponseField name="total_cost" type="number">
      Total cost across all sessions
    </ResponseField>
    <ResponseField name="total_tokens" type="number">
      Total tokens used
    </ResponseField>
    <ResponseField name="avg_requests_per_session" type="number">
      Average requests per session
    </ResponseField>
    <ResponseField name="avg_cost_per_session" type="number">
      Average cost per session
    </ResponseField>
    <ResponseField name="avg_duration" type="number">
      Average session duration in seconds
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.helicone.ai/v1/session/metrics/query \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "nameContains": "",
    "timezoneDifference": 0,
    "pSize": "p95",
    "useInterquartile": false
  }'
```

```python Python
import requests

url = "https://api.helicone.ai/v1/session/metrics/query"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

payload = {
    "nameContains": "",
    "timezoneDifference": 0,
    "pSize": "p95",
    "useInterquartile": False
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()
```

