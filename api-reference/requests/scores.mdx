---

## Overview

Attach numeric or boolean scores to requests for tracking quality metrics, evaluation results, or custom scoring.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Path Parameters

<ParamField path="requestId" type="string" required>
  The unique identifier of the request
</ParamField>

## Request Body

<ParamField body="scores" type="object" required>
  Object containing score key-value pairs. Values must be integers or booleans.
  
  Example:
  ```json
  {
    "accuracy": 95,
    "relevance": 87,
    "contains_pii": false,
    "is_helpful": true
  }
  ```
</ParamField>

## Response

<ResponseField name="data" type="null">
  Returns null on success
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.helicone.ai/v1/request/req_abc123/score \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "scores": {
      "accuracy": 95,
      "relevance": 87,
      "is_helpful": true
    }
  }'
```

```python Python
import requests

url = "https://api.helicone.ai/v1/request/req_abc123/score"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
payload = {
    "scores": {
        "accuracy": 95,
        "relevance": 87,
        "is_helpful": True
    }
}

response = requests.post(url, json=payload, headers=headers)
```

```javascript JavaScript
const response = await fetch(
  'https://api.helicone.ai/v1/request/req_abc123/score',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      scores: {
        accuracy: 95,
        relevance: 87,
        is_helpful: true,
      },
    }),
  }
);
```

</RequestExample>

## Score Requirements

<Note>
  - All numeric scores must be **integers** (whole numbers)
  - Boolean values (`true`/`false`) are automatically converted to 1/0
  - Float values will cause an error
</Note>

## Use Cases

- Track LLM evaluation metrics (accuracy, relevance, coherence)
- Record human feedback scores
- Monitor safety checks (toxicity, PII detection)
- Measure custom quality metrics
- Store A/B test performance indicators
