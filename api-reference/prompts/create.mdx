---
title: 'Create Prompt'
api: 'POST https://api.helicone.ai/v1/prompt/create'
description: 'Create a new prompt template with versioning'
---

## Overview

Create a new prompt template in Helicone for version control and management.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="userDefinedId" type="string" required>
  Unique identifier for the prompt (e.g., "customer-support-v1")
</ParamField>

<ParamField body="prompt" type="object" required>
  Prompt template object in Helicone template format
  
  Example:
  ```json
  {
    "model": "gpt-4",
    "messages": [
      {
        "role": "system",
        "content": "You are a helpful assistant."
      }
    ]
  }
  ```
</ParamField>

<ParamField body="metadata" type="object" required>
  Additional metadata for the prompt
  
  Example:
  ```json
  {
    "description": "Customer support prompt",
    "tags": ["support", "production"]
  }
  ```
</ParamField>

## Response

<ResponseField name="data" type="object">
  Created prompt information
  
  <Expandable title="prompt response">
    <ResponseField name="id" type="string">
      Unique prompt ID
    </ResponseField>
    <ResponseField name="prompt_version_id" type="string">
      ID of the created prompt version
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

```python Python
import requests

url = "https://api.helicone.ai/v1/prompt/create"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

payload = {
    "userDefinedId": "customer-support-v1",
    "prompt": {
        "model": "gpt-4",
        "messages": [
            {
                "role": "system",
                "content": "You are a helpful customer support assistant."
            }
        ]
    },
    "metadata": {
        "description": "Main customer support prompt",
        "tags": ["support", "production"]
    }
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()
```
