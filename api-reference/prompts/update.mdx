---
title: 'Update Prompt'
api: 'PATCH https://api.helicone.ai/v1/prompt/{promptId}/user-defined-id'
description: 'Update a prompt user-defined identifier'
---

## Overview

Update the user-defined ID of an existing prompt.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Path Parameters

<ParamField path="promptId" type="string" required>
  The unique identifier of the prompt
</ParamField>

## Request Body

<ParamField body="userDefinedId" type="string" required>
  New user-defined identifier for the prompt
</ParamField>

## Response

<ResponseField name="data" type="null">
  Returns null on success
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

<RequestExample>

```bash cURL
curl --request PATCH \
  --url https://api.helicone.ai/v1/prompt/prompt_abc123/user-defined-id \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "userDefinedId": "customer-support-v2"
  }'
```

```python Python
import requests

url = "https://api.helicone.ai/v1/prompt/prompt_abc123/user-defined-id"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
payload = {
    "userDefinedId": "customer-support-v2"
}

response = requests.patch(url, json=payload, headers=headers)
```

```javascript JavaScript
const response = await fetch(
  'https://api.helicone.ai/v1/prompt/prompt_abc123/user-defined-id',
  {
    method: 'PATCH',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      userDefinedId: 'customer-support-v2',
    }),
  }
);
```

</RequestExample>
