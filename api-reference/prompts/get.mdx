---
title: 'Get Prompt'
api: 'POST https://api.helicone.ai/v1/prompt/{promptId}/query'
description: 'Retrieve a specific prompt with version history'
---

## Overview

Get detailed information about a prompt including its versions and usage statistics.

## Authentication

<ParamField header="Authorization" type="string" required>
  Your Helicone API key. Format: `Bearer YOUR_API_KEY`
</ParamField>

## Path Parameters

<ParamField path="promptId" type="string" required>
  The unique identifier of the prompt
</ParamField>

## Request Body

<ParamField body="timeFilter" type="object" required>
  Time range for usage statistics
  
  <Expandable title="time filter properties">
    <ResponseField name="start" type="string" required>
      Start time in ISO 8601 format
    </ResponseField>
    <ResponseField name="end" type="string" required>
      End time in ISO 8601 format
    </ResponseField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="data" type="object">
  Prompt object
  
  <Expandable title="prompt properties">
    <ResponseField name="id" type="string">
      Unique prompt ID
    </ResponseField>
    <ResponseField name="user_defined_id" type="string">
      User-defined identifier
    </ResponseField>
    <ResponseField name="description" type="string">
      Prompt description
    </ResponseField>
    <ResponseField name="pretty_name" type="string">
      Display name
    </ResponseField>
    <ResponseField name="major_version" type="number">
      Current major version number
    </ResponseField>
    <ResponseField name="latest_version_id" type="string">
      ID of the latest version
    </ResponseField>
    <ResponseField name="latest_model_used" type="string">
      Most recently used model
    </ResponseField>
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
    <ResponseField name="last_used" type="string">
      ISO 8601 timestamp of last usage
    </ResponseField>
    <ResponseField name="versions" type="array">
      Array of version IDs
    </ResponseField>
    <ResponseField name="metadata" type="object">
      Custom metadata
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="error" type="string">
  Error message if request failed
</ResponseField>

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.helicone.ai/v1/prompt/prompt_abc123/query \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "timeFilter": {
      "start": "2024-01-01T00:00:00Z",
      "end": "2024-03-02T23:59:59Z"
    }
  }'
```

```python Python
import requests

url = "https://api.helicone.ai/v1/prompt/prompt_abc123/query"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

payload = {
    "timeFilter": {
        "start": "2024-01-01T00:00:00Z",
        "end": "2024-03-02T23:59:59Z"
    }
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()
```

