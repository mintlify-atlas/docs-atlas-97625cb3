## Introduction

Helicone provides seamless integration with leading AI providers and frameworks, offering comprehensive observability and monitoring for your AI applications. Whether you're using OpenAI, Anthropic, LangChain, or other popular tools, Helicone has you covered.

## AI Gateway vs Legacy Integrations

### AI Gateway (Recommended)

The Helicone AI Gateway is the modern, unified approach to integrate with 100+ AI models:

- **Single API** for all providers
- **Automatic fallbacks** if a provider goes down
- **Intelligent routing** based on cost, latency, or custom rules
- **Zero markup** on model costs
- **Unified observability** across all providers

<Card title="Get Started with AI Gateway" icon="rocket" href="/gateway/overview">
  Start using the unified API for 100+ models in 2 minutes
</Card>

### Legacy Provider Integrations

For existing applications built with provider-specific SDKs, Helicone offers proxy-based integrations:

- Simple baseURL change to route through Helicone
- Keep using your existing SDK code
- Provider-specific endpoints (e.g., `oai.helicone.ai` for OpenAI)

<Warning>
  Legacy integrations are maintained but not actively developed. For new projects, use the AI Gateway.
</Warning>

## Integration Features

All Helicone integrations provide:

### Observability
- **Request logging** - Track all API calls with full request/response data
- **Cost tracking** - Monitor spending across providers and models
- **Latency monitoring** - Measure response times and performance
- **Error tracking** - Capture and debug failed requests

### Advanced Features
- **Custom properties** - Tag requests with metadata for filtering
- **Sessions** - Group related requests into conversations
- **User tracking** - Associate requests with end users
- **Caching** - Reduce costs and latency with semantic caching
- **Rate limiting** - Control API usage per user or endpoint

### Analytics
- **Usage dashboards** - Visualize API usage patterns
- **Cost analytics** - Track spending trends and optimize costs
- **Performance metrics** - Monitor latency, throughput, and success rates
- **User analytics** - Understand user behavior and engagement
