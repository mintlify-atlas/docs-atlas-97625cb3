This guide will get you up and running with the AI Gateway in minutes. You'll learn how to access 100+ models through a single API.

## Prerequisites

- A Helicone account ([sign up here](https://helicone.ai/signup))
- Node.js, Python, or any HTTP client
- 2 minutes of your time

## Step 1: Get Your API Key

<Steps>
  <Step title="Sign Up">
    Create a free account at [helicone.ai/signup](https://helicone.ai/signup)
  </Step>
  
  <Step title="Get API Key">
    Your API key is available immediately in the dashboard at [helicone.ai/settings](https://helicone.ai/settings)
  </Step>
  
  <Step title="Add Credits (Optional)">
    For PTB (Pay Through Billing), add credits at [helicone.ai/credits](https://helicone.ai/credits). Not needed if using BYOK.
  </Step>
</Steps>

## Step 3: Make Your First Request

<Tabs>
  <Tab title="TypeScript/JavaScript">
    ```typescript
    import OpenAI from "openai";

    const client = new OpenAI({
      baseURL: "https://ai-gateway.helicone.ai",
      apiKey: process.env.HELICONE_API_KEY,
    });

    async function main() {
      const response = await client.chat.completions.create({
        model: "gpt-4o-mini",
        messages: [
          {
            role: "user",
            content: "Explain quantum computing in simple terms"
          }
        ],
      });

      console.log(response.choices[0].message.content);
    }

    main();
    ```
  </Tab>
  
  <Tab title="Python">
    ```python
    from openai import OpenAI

    client = OpenAI(
        base_url="https://ai-gateway.helicone.ai",
        api_key="YOUR_HELICONE_API_KEY"
    )

    response = client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[
            {
                "role": "user",
                "content": "Explain quantum computing in simple terms"
            }
        ],
    )

    print(response.choices[0].message.content)
    ```
  </Tab>
  
  <Tab title="cURL">
    ```bash
    curl https://ai-gateway.helicone.ai/v1/chat/completions \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer YOUR_HELICONE_API_KEY" \
      -d '{
        "model": "gpt-4o-mini",
        "messages": [
          {
            "role": "user",
            "content": "Explain quantum computing in simple terms"
          }
        ]
      }'
    ```
  </Tab>
</Tabs>

All requests are automatically logged. View them at:

- **Requests**: [helicone.ai/requests](https://helicone.ai/requests)

## Using BYOK (Bring Your Own Keys)

To use your own provider API keys instead of PTB:

<Steps>
  <Step title="Add Provider Keys">
    Go to [helicone.ai/settings/keys](https://helicone.ai/settings/keys)
  </Step>
  
  <Step title="Configure Providers">
    Add API keys for OpenAI, Anthropic, Google, or other providers
  </Step>
  
  <Step title="Use the Gateway">
    The gateway automatically prioritizes your keys (BYOK) over PTB:
    
    ```typescript
    // Same code - gateway uses your keys automatically
    const response = await client.chat.completions.create({
      model: "claude-sonnet-4",
      messages: [{ role: "user", content: "Hello!" }],
    });
    ```
  </Step>
</Steps>

## Streaming Responses

The gateway fully supports streaming:

<Tabs>
  <Tab title="TypeScript/JavaScript">
    ```typescript
    const stream = await client.chat.completions.create({
      model: "gpt-4o-mini",
      messages: [{ role: "user", content: "Write a haiku about coding" }],
      stream: true,
    });

    for await (const chunk of stream) {
      const content = chunk.choices[0]?.delta?.content || "";
      process.stdout.write(content);
    }
    ```
  </Tab>
  
  <Tab title="Python">
    ```python
    stream = client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{"role": "user", "content": "Write a haiku about coding"}],
        stream=True,
    )

    for chunk in stream:
        content = chunk.choices[0].delta.content or ""
        print(content, end="")
    ```
  </Tab>
</Tabs>

## Configuration Headers

Customize gateway behavior with request headers:

```typescript
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "https://ai-gateway.helicone.ai",
  apiKey: process.env.HELICONE_API_KEY,
  defaultHeaders: {
    // Tag requests for filtering
    "Helicone-Property-Environment": "production",
    "Helicone-Property-User": "user-123",
    
    // Enable caching (see /gateway/caching)
    "Helicone-Cache-Enabled": "true",
    
    // Rate limiting (see /gateway/rate-limiting)
    "Helicone-RateLimit-Policy": "100;w=60;s=user",
  },
});
```

## Error Handling

Handle errors gracefully:

```typescript
try {
  const response = await client.chat.completions.create({
    model: "gpt-4o-mini",
    messages: [{ role: "user", content: "Hello!" }],
  });
  
  console.log(response.choices[0].message.content);
} catch (error) {
  if (error.status === 429) {
    console.error("Rate limited or insufficient credits");
  } else if (error.status === 401) {
    console.error("Invalid API key");
  } else {
    console.error("Request failed:", error.message);
  }
}
```
